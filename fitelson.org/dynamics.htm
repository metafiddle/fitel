Deliberational Dynamics

Below, you'll find links to four Mathematica package files and notebooks of mine which provide various functions for performing both numerical simulations and theoretical stability analyses of 2x2 normal form games under a variety of deliberational (viz., evolutionary) dynamics, in both discrete and continuous time. All dynamical models are taken from Brian Skyrms 's book The Dynamics of Rational Deliberation.

For those of you who don't have Mathematica, you can download Wolfram's free MathReader. Or, you can download an the Adobe Acrobat PDF (which requires Adobe Acrobat Reader) versions of the notebooks instead. Of course, with the PDF versions, you'll only be able to view/print the notebooks; you won't be able to interact with them.

  • discr2x2.m -- Mathematica 2.2 package file which contains various functions for both numerical simulations and theoretical stability analyses of the evolutionary dynamics of 2x2 normal form games in discrete time.
    • discr2x2.nb -- Mathematica 3.0 notebook which explains how to use the various functions defined in the package discr2x2.m. Click here for a PDF version of this notebook (requires Acrobat Reader) .

  • cont2x2.m -- Mathematica 2.2 package file which contains various functions for both numerical simulations and theoretical stability analyses of the evolutionary dynamics of 2x2 normal form games in continuous time.
    • cont2x2.nb -- Mathematica 3.0 notebook which explains how to use the various functions defined in the package cont2x2.m. Click here for a PDF version of this notebook (requires Acrobat Reader).

  • dynamics.zip -- ZIP archive containing all 4 of the above Mathematica files.
